**Project Running Instructions**

1. Download and Install Xampp
https://www.apachefriends.org/download.html

2. Download Backend PHP
https://drive.google.com/file/d/11tH0cx1OvxI29OHOoYYDrANAr9OcZkZQ/view?usp=sharing

- Unzip the file
- Move the extracted folder to xampp/htdocs folder
- Start xampp

3. Clone the repository
git clone https://github.com/nhridoy/xpeed-task.git

4. Install Dependencies
npm install

5. Start the Project
npm start
